CSS Selectors & Specificity

These are the different CSS specificity selectors:
Type Description HTML syntaxis CSS syntaxis
Element tag

Selects a group of HTML elements by their element tag as in HTML

<tag>

tag {
... }
Class

Assigns a custom name to an element which can be assigned to multiple elements

<tag class="XXXXX">

.XXXXX {
...
}
ID

Assigns a custom name to an element which can be assigned to only one element

<id="XXXXX">

#"XXXXX" {
...
}
!important

CSS exclusive assignation which overrides any other rule of specificity in CSS
